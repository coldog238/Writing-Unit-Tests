# Лабораторная работа №4: Модульное тестирование

## Описание
Реализация модульных тестов для функций вычисления площади и периметра прямоугольника.

## Функции
- `square(a, b)` - вычисление площади прямоугольника
- `perimetr(a, b)` - вычисление периметра прямоугольника

## Запуск тестов
```bash
python -m unittest test_geometry.py
